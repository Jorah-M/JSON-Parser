# JSON Parser

Скрипт на Node.js, который читает произвольный текст (например, список покупок), отправляет его в Groq API и сохраняет структурированный результат в JSON.

## Установка и запуск

```bash
npm install
cp .env.example .env
# Вставьте ваш ключ в .env: GROQ_API_KEY=sk-...
npm start
```

### Свой входной файл

```bash
npm start -- my-list.txt
```

По умолчанию читается `input.txt`.

## Пример входного текста (`input.txt`)

```
Нужно купить: молоко 2 литра за 89 рублей, хлеб белый — две буханки,
яблоки примерно килограмм (стоят около 120р), кофе растворимый 1 банка
и ещё не забыть про стиральный порошок.
```

## Пример результата

Вывод в консоль:

```
Результат:
┌─────────┬─────────────────────┬──────────────┬───────────┐
│ (index) │ Товар               │ Количество   │ Цена      │
├─────────┼─────────────────────┼──────────────┼───────────┤
│    0    │ 'молоко'            │ '2 литра'    │ '89 руб.' │
│    1    │ 'хлеб белый'        │ '2 буханки'  │ 'unknown' │
│    2    │ 'яблоки'            │ '1 кг'       │ '120 руб.'│
│    3    │ 'кофе растворимый'  │ '1 банка'    │ 'unknown' │
│    4    │ 'стиральный порошок'│ 'unknown'    │ 'unknown' │
└─────────┴─────────────────────┴──────────────┴───────────┘

Результат сохранён в output_2026-03-20T14-32-00.json
```

Файл `output_<timestamp>.json`:

```json
{
  "items": [
    { "name": "молоко", "quantity": "2 литра", "price": "89 рублей" },
    { "name": "хлеб белый", "quantity": "2 буханки", "price": "unknown" },
    { "name": "яблоки", "quantity": "1 кг", "price": "120 рублей" },
    { "name": "кофе растворимый", "quantity": "1 банка", "price": "unknown" },
    { "name": "стиральный порошок", "quantity": "unknown", "price": "unknown" }
  ]
}
```

## Структура проекта

```
├── index.js          # Главный скрипт

├── package.json
├── .env.example      # Шаблон переменных окружения
├── .gitignore
```
